CSS Flexbox

To use flexbox, first set the display property of the parent element to flex.

(In the example to the right, our parent element has a class name of container.)

See it in Popcode! 
.container {
display: flex;
}

Aligning Child Elements

To align the child elements to a specific side, use the justify-content property with one of the following values: flex-start, center, flex-end, space-between, space-around

For example, the CSS code below will align the child elements (colored boxes) to the center of the parent element with the class name of container.

See it in Popcode! 
.container {
display: flex;
justify-content: center;
}

Flexbox Columns

Other than aligning child elements, flexbox can also be used to create columns.
Simply give each child element a width and now you have a columned layout!

See it in Popcode! 
.section {
display: flex;
}

.left {
width: 30%;
}

.right {
width: 70%;
}
